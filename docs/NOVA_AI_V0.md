# NOVA Core AI v0

## Purpose

NOVA Core AI v0 provides a reliable, explainable decision-intelligence layer for the SolarHub demo.

This is not full autonomous AI. It is a deterministic, auditable AI-shaped workflow that supports forecasts, recommendations, simulations, and impact scoring.

## Services

### Forecast Service

Generates deterministic forecasts for:

- Water demand
- Solar production
- Battery state
- Weather risk
- Yield risk
- Carbon impact

### Recommendation Service

Generates operator-approved recommendations with:

- Title
- Description
- Priority
- Confidence
- Expected impact
- Explanation
- Status

### Scenario Simulation Service

Compares baseline vs optimized scenarios.

### Impact Scoring Service

Calculates expected water, energy, carbon, yield, and resilience impact.

### AI Assistant Explanation Layer

Explains what is happening, why it matters, what will happen next, and what the operator should do.

## SolarHub Demo Scenario

Heatwave forecast → increased irrigation demand → battery/pump coordination → evening irrigation recommendation → measured water, energy, carbon, and resilience impact.
