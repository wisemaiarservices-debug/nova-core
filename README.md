# NOVA Core

NOVA Core is the shared platform foundation for **NOVA OS**, the AI Operating System for Sustainable Infrastructure by NeoHaven AI Urban Labs.

## Purpose

NOVA Core provides the shared services, data model, AI workflow, and architecture used by:

- NOVA Energy, powered by NeoGrid
- NOVA Agriculture, powered by NeoAgro
- NOVA Infrastructure, powered by NeoCell
- NOVA Twin
- NOVA Intelligence

## Shared Domains

- Organizations
- Sites
- Assets
- Sensors
- Telemetry
- Digital Twins
- Knowledge Graph relationships
- Forecasts
- Recommendations
- Recommendation results
- Optimization runs
- Reports
- Audit logs

## Core Workflow

Observe → Analyze → Forecast → Recommend → Simulate → Optimize → Measure Impact

## SolarHub Role

For the SolarHub demo, NOVA Core should provide a deterministic, reliable AI v0 layer:

- Forecast service
- Recommendation service
- Scenario simulation service
- Impact scoring service
- AI assistant explanation layer

Version 0 can be rules-based and deterministic. Reliability is more important than model complexity for September.
