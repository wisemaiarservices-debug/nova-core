NeoHaven AI Urban Labs is building NOVA OS, an AI Operating System for Sustainable Infrastructure.

## NOVA Core Purpose

NOVA Core provides shared platform services for:

- NOVA Energy / NeoGrid
- NOVA Agriculture / NeoAgro
- NOVA Infrastructure / NeoCell
- NOVA Twin
- NOVA Intelligence

## Engineering Rules

- Do not generate toy examples.
- Preserve compatibility with NeoAgro, NeoGrid, and NeoCell.
- Build shared services incrementally.
- Prefer deterministic AI v0 for SolarHub demo reliability.
- Every feature should include architecture, database/API impact, tests, and documentation.
- NOVA recommends actions for operator approval.
